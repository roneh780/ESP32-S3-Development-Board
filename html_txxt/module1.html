1. Introduction

1.1 ESP32-S3 Series Overview

ESP32-S3 is a new generation of low-power, high-performance Wi-Fi + Bluetooth® LE MCU chip launched by Espressif Systems, designed for artificial intelligence of things (AIoT), edge computing, smart home, human-computer interaction, and other application scenarios. Compared to the previous generation ESP32, ESP32-S3 has significant improvements in computing power, AI acceleration capability, connection security, and storage expandability.

The ESP32-S3 series includes multiple models, mainly differing in built-in Flash and PSRAM capacities. The model used in this tutorial is ESP32-S3 N16R8, with the following naming convention:

  • N16: Built-in 16 MB serial Flash (Nor Flash), used for storing program code and file system.
  • R8: Built-in 8 MB PSRAM (pseudo-static random access memory), which can significantly expand runtime memory, suitable for processing large memory tasks such as images, audio buffers, and complex data structures.

1.2 ESP32-S3 N16R8 Core Features

Processor and Computing Capability

  • Dual-core Xtensa® LX7 processor: Maximum clock speed up to 240 MHz, each core runs independently, supporting symmetric multiprocessing (SMP) and task allocation.
  • Vector instruction set: Provides SIMD (Single Instruction Multiple Data) acceleration, significantly improving digital signal processing (DSP) and neural network inference efficiency.
  • TensorFlow Lite Micro support: Can directly run lightweight machine learning models on the chip, enabling edge AI applications such as voice wake-up and image recognition.

Storage Configuration

Storage Type Capacity Function
Built-in Flash 16 MB Program storage, file system, web resources
Built-in PSRAM 8 MB Runtime dynamic memory, supports camera frame buffer, large arrays, graphics cache, etc.

Note: The large-capacity PSRAM of N16R8 makes ESP32-S3 more comfortable when processing OV2640/OV5640 camera data and driving high-resolution displays (such as ILI9341, ST7789), without system crashes due to insufficient memory.

Wireless Communication

  • 2.4 GHz Wi-Fi 802.11 b/g/n:
    • Supports Station (client), SoftAP (hotspot), and Station + SoftAP mixed modes.
    • Supports WPA3-Personal encryption for improved security.
  • Bluetooth 5.0:
    • Supports both classic Bluetooth (BR/EDR) and Bluetooth Low Energy (BLE).
    • Supports BLE Mesh networking, suitable for large-scale device networks.

Rich Peripheral Interfaces

ESP32-S3 N16R8 provides up to 45 programmable GPIOs, with common interfaces as follows:

Interface Type Quantity Main Functions
ADC (12-bit) 2 (20 channels total) Analog sensor acquisition (light sensors, potentiometers, microphones, etc.)
DAC (8-bit) 2 Analog voltage output (audio, analog signal generation)
Touch sensors 14 channels Capacitive touch buttons, can penetrate insulating panels
UART 4 Serial communication (GPS, Bluetooth modules, debugging)
SPI 3 High-speed peripherals (displays, SD cards, sensors)
I2C 2 Low-speed sensors (temperature/humidity, IMU, OLED)
I2S 2 Digital audio input/output (microphones, speakers)
SDIO Host 1 Connect SD cards or SDIO devices
USB OTG 1 Configurable as USB host or device (supports keyboards, U disks, MIDI, etc.)
PWM Any GPIO Up to 16 independent PWM channels through LEDC controller
External interrupts All GPIOs Supports edge/level triggering, quick response to external events

Artificial Intelligence Acceleration Features

ESP32-S3 is Espressif's first general-purpose MCU with a built-in vector instruction set. Combined with TensorFlow Lite Micro, it can complete the following tasks locally:

  • Keyword wake-up (e.g., local recognition of "Hey Google")
  • Gesture recognition (using accelerometer/gyroscope data)
  • Image classification (with camera, such as face detection)
  • Abnormal sound detection (industrial equipment fault warning)

No need to upload raw data to the cloud, reducing latency and protecting privacy.

Security Features

ESP32-S3 provides industry-leading IoT security features:

  • Hardware encryption accelerator: AES-128/256, SHA-2 (256/384/512), RSA large number operations.
  • Secure boot: Ensures only signed firmware runs.
  • Flash encryption: Prevents external reading of firmware content.
  • Digital signature: Peripheral private keys cannot be read out, can be used for device identity authentication.
  • HMAC module: Used to generate message authentication codes.

1.3 Typical Application Scenarios

Based on the above features, ESP32-S3 N16R8 is particularly suitable for the following projects:

Application Field Example Projects
Smart Home Multi-sensor environment monitoring station, smart speakers, Wi-Fi/BLE lighting control
Human-Computer Interaction Voice assistants, gesture-controlled cars, touch panels
Edge AI Facial recognition access control, mask detection, abnormal sound monitoring
IoT Gateway BLE Mesh gateway, Modbus to MQTT gateway
Multimedia Processing Network cameras, MP3 audio players, electronic photo albums
Industrial Control Data acquisition terminals, remote relay control, motor speed regulation

1.4 Development Environment and Arduino Support

ESP32-S3 is fully compatible with the Arduino ecosystem. By installing Espressif's official esp32 board support package (version 3.0.x or higher), you can use Arduino IDE for development. This tutorial will be based on Arduino IDE, starting from scratch to configure the ESP32-S3 N16R8 development environment, and will cover:

  • GPIO digital/analog input/output
  • Interrupts, timers, PWM, ADC
  • Serial communication (UART, I2C, SPI)
  • Wi-Fi network and web server
  • Bluetooth (BLE) communication
  • Camera and display module driving
  • Comprehensive practical projects

Through the study of this tutorial, you will be able to skillfully use ESP32-S3 N16R8 to complete development from simple control to complex AIoT systems.
